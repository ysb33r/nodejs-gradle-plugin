[Groovy] Class NpmTask

            • org.ysb33r.gradle.nodejs.tasks.NpmTask

            • Ability to execute any NPM command with parameters.

              The task contains an NPM extension, which by default is setup to look at the global NPM extension. It allows overriding on a per-task basis of NPM configuration.

                • @java.lang.Override void exec()

                  Runs NPM task against an internal execution specification.

                  If a failure occurs and isIgnoreExitValue is not set an exception will be raised.

                  Sets the npm_config_userconfig and npm_config_globalconfig before running from either the local or global npm extension (in that order). It will also configure a node executable using either the local or gloval nodejs extension.
