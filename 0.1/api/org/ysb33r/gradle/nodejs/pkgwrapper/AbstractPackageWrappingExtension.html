[Groovy] Class AbstractPackageWrappingExtension<R extends AbstractPackageResolver>

    • org.ysb33r.gradle.nodejs.pkgwrapper.AbstractPackageWrappingExtension

    • A base class that can be used to wrap specific NPM packages as features in Gradle, thereby allowing for configuraton both a project or task level.
