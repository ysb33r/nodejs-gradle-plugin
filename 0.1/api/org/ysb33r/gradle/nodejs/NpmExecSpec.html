[Groovy] Class NpmExecSpec

            • org.ysb33r.gradle.nodejs.NpmExecSpec

            • Specification for running an NPM command via npm-cli.js

              For simplicity Gradle executes npm-cli.js directly rather than use yet another indirection of the npm shell script.
