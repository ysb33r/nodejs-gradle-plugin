Package: org.ysb33r.gradle.nodejs

[Groovy] Class GulpExtension

      • org.ysb33r.gradle.nodejs.GulpExtension

      • Extension that allows for setting of Gulp configuration at a project or task level. If no executable is set the default will be to install GulpExtension.GULP_DEFAULT for a project extension. In the case of a task extension it will default to the project extension's settings.
